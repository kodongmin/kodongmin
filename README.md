## 🧑‍💻 About Me

안녕하세요! 저는 **백엔드 서버 아키텍처 설계와 클라우드 환경 구축**에 집중하고 있는 개발자 **고동민**입니다.  
단순히 코드를 작성하는 것을 넘어서, **“왜 이렇게 동작하는가”** 를 이해하고 설계의 의미를 고민합니다.  

현재 진행 중인 `Nebula` 프로젝트를 통해  
- **FastAPI 기반 로컬 클라이언트**와  
- **Spring Boot + AWS EC2 백엔드 서버** 간의 연동을 구축하며,  
파일 메타데이터 처리와 OpenAI API 연동을 통한 자동화 시스템을 구현하고 있습니다.  

이외에도 OS, 보안, 클라우드 컴퓨팅 수업을 통해 시스템의 근본적인 원리를 탐구하며,  
안정성과 확장성을 모두 갖춘 개발자를 목표로 하고 있습니다.
